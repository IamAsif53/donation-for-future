The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document (like an element, attribute, or text). Using the DOM, developers can access, modify, or delete elements and content on a web page dynamically via languages like JavaScript. The DOM allows the webpage to update without reloading.

I can select an element from the DOM using JavaScript in various ways:

By ID:

document.getElementById('elementId');

By Class Name:

document.getElementsByClassName('className');

By Tag Name:

const elements = document.getElementsByTagName('p');
